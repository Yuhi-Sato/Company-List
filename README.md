# Company-List
今までは、スプレッドシートやエクセルやテキストなど、部署や個人で管理していた企業情報を、1つにまとめた検索システム

# 概要
インターン選考の技術課題として、作成しました（技術課題概要.txt参照）。

「Company-List」には以下の機能があります。
* ユーザー認証機能
* ユーザー権限管理
* 企業情報の閲覧・作成・編集・削除
* 企業会計情報の閲覧・作成・編集・削除
* 売上・利益のチャート閲覧
